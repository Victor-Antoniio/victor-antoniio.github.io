Padaria do Bairro - Website

Descrição do Projeto:
Este é um projeto de website simples para a "Padaria do Bairro", desenvolvido inteiramente em HTML. O site tem como objetivo apresentar a padaria, seus produtos, horários de funcionamento e oferecer diferentes canais de interação com os clientes através de formulários.

Estrutura do Projeto:

O projeto é composto pelos seguintes arquivos HTML principais:

- index.html: A página inicial do website, contendo informações sobre a padaria (história, missão, cardápio, fotos do ambiente, diferenciais, localização e horários de funcionamento). Esta página também serve como HUB para navegar para os diferentes formulários.
- Formulário-Cadastro-Newsletter.html: Um formulário para os clientes se cadastrarem na newsletter da padaria, coletando dados pessoais, endereço e preferências de produtos.
- Formulário-Feedback.html: Um formulário onde os clientes podem deixar seu feedback sobre produtos, serviços, atendimento, ambiente e preços da padaria.
- Formulário-de-Contato.html: Um formulário geral para os clientes entrarem em contato com a padaria para informações, reclamações, sugestões, elogios ou pedidos especiais.
- Formulário-pedidos-especiais.html: Um formulário dedicado a clientes que desejam fazer pedidos especiais de bolos, tortas ou salgados, com opções de sabor, tamanho e data de entrega.

Além dos arquivos HTML, o projeto utiliza recursos multimídia locais:

- Imagens: Fotos dos produtos da padaria, do ambiente interno e ícones (como newsletter e telefone).
- Vídeos: Um vídeo do ambiente externo da padaria.

Como Visualizar o Projeto:

Para visualizar o projeto, siga os passos abaixo:

1. Baixe ou clone todos os arquivos do projeto para uma pasta em seu computador.
2. Abra o arquivo `index.html` em qualquer navegador de internet moderno (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.).

Navegação:

A partir da página `index.html`, você pode navegar para os diferentes formulários clicando nos botões correspondentes no cabeçalho:
- "Entre em Contato Conosco" (Formulário-de-Contato.html)
- "Faça um Pedido Especial" (Formulário-pedidos-especiais.html)
- "Como foi a sua Experiência?" (Formulário-Feedback.html)
- "Cadastre-se" (Formulário-Cadastro-Newsletter.html)

Cada formulário possui um botão "Voltar a Tela Inicial" no cabeçalho para retornar ao `index.html`.

Observações:

- Este projeto é desenvolvido apenas com HTML, sem o uso de CSS para estilização avançada ou JavaScript para funcionalidades dinâmicas. A estilização é básica e feita diretamente no HTML (ex: `bgcolor` para cor de fundo e `style` para botões).
- Todos os recursos de imagem, ícones e vídeos são locais e devem estar nas pastas `imagens/icones`, `imagens/Logo`, `imagens/Produtos`, `imagens/Ambiente` e `videos` respectivamente, dentro do diretório raiz do projeto para serem exibidos corretamente.
